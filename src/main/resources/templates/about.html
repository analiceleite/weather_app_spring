About Weather App

The Weather App provides real-time weather data based on geographic coordinates (latitude and longitude). Users can enter their location to receive accurate temperature readings.

Features:

  • Real-time weather updates
  • Simple and user-friendly interface
  • Responsive design for mobile and desktop

Technologies Used:

  • Thymeleaf for server-side rendering
  • Tailwind CSS for styling
  • Java for backend development
